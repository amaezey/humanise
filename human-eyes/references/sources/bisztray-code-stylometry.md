## Summary
Code-stylometry paper for attributing C programs generated by LLMs to their source model. It introduces CodeT5-Authorship and LLM-AuthorBench, a benchmark of 32,000 compilable C programs generated by eight LLMs.

## Main insights
- Evidence that model-source attribution can work in generated code.
- The abstract is not prose evidence and should not justify human-eyes writing patterns.
- Useful only as a code-stylometry boundary case alongside prose-specific idiosyncrasy papers.

## Evidence and claims to extract
- Benchmark: 32,000 compilable C programs from eight state-of-the-art LLMs.
- Reported performance: 97.56% binary accuracy distinguishing closely related GPT-4.1 and GPT-4o generated C programs; 95.40% multi-class attribution among Gemini 2.5 Flash, Claude 3.5 Haiku, GPT-4.1, Llama 3.3, and DeepSeek-V3.
- Scope limitation: code, specifically C programs, not natural-language prose.
